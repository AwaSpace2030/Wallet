# Wallet
 

**Wallet** is a minimal expense tracking application built for simplicity and clarity.

---

## Live Demo
Try the app live here: [Wallet App](https://wallet1-app.netlify.app/)

---

## Project Overview
Wallet is a personal finance React web application that helps users track their daily expenses in a clean and intuitive way. The focus of this project is to remove unnecessary complexity and provide a smooth user experience.

---

## Features
- Track daily expenses
- Categorize expenses
- Monthly expense dashboard
- Visual charts & statistics
- Simple and clean interface
- Secure authentication
- Responsive design
- Cloud data storage with Firebase

---

## Tech Stack
- React
- React Router
- Firebase (Authentication & Cloud Firestore)
- CSS Modules
- Framer Motion

---

## Purpose
This project was created as a learning experience and a foundation for future features and improvements.

---

## Future Improvements

- Export reports to PDF/CSV
- Notifications & reminders
- Income tracking (ability to log and track income)
